Open...

Click on Open to open either a coordinate data file in XYZ-format, a list of coordinate data files, or a view parameter file.

When opening a coordinate data file, the file browser will sport an extra button labeled Generate File List. The button is associated with a convenience function that wil create a file list (see section 4.2) including all coordinate files in the current directories. Coordinate files are recognized if their filename contains the suffix .xyz.
