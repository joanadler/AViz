Command Line Arguments

AViz accepts standard X11 command line arguments. In addition, the following command line arguments given in table 3.1 are defined.


Table 1: AViz Command line arguments
-ar Start auto rotation
-as Start auto spinning
-at Start auto tilting
-az Start auto zooming
-asm Start in auto snap mode
-fl [listfile] Start using specified file list
-rm [atom|spin|liquidcrystal|polymer|pore] Set render mode
-rs [dots|lines|cubes|cones|cylinders|spheres] Set render style
-rq [low|high|final] Set render quality
-si [percentage] Set render size in percentage
-snap Take a screen shot of the
  rendering at startup
-snapq Take a screen shot of the
  rendering at startup and exit
-v(ersion) Print version number and exit
-vpm [viewParamfile] Use view parameters in specified
  file at startup
-watch [XYZfile] Update rendering upon changes in
  specifed coordinate file
-license Print GNU license applying to this
  software
-distribution Print GNU regulations under applying
  to this software
-usage|-h(elp) Print this message and exit
