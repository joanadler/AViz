Watch XYZ File

The Watch XYZ File button causes watching of the currently rendered coordinate file: whenever the file changes, the graphics display is updated, showing the current data. File changes are checked every 10 seconds.

If the AutoSnap button is depressed, you will obtain a PNG snapshot upon every update. This may be a convenient way to generate a visualization of the dynamics in your simulation; see also section 8.1.
