Watching a File

In some situations one would like to monitor an on-going simulation. Here the -watch option may be of use. When AViz is started up as in 

aviz -watch wedgeSim.xyz

it will render the data set in the specified file. Every 10 seconds or so, AViz will re-examine the file. If it has been modified in the mean time, AViz will automatically re-read the file and udpate the rendering. If the Auto Snap button is depressed, a sequence of PNG images is generated as a byproduct.
