Stretch...

This entry pops up a window with three text windows, one for each coordinate. In each window a factor can be specified with which all coordinates are multiplied during rendering. A factor greater then 1 means that the rendered structure is extended in the direction at hand.

Buttons next to the text windows can be used to reset the entries to their default values of 1.

The setting is active after Apply or Done is pressed.
