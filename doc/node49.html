File List Dialog

The file list dialog pops up when you press the File List... entry in the File menu.

The current file list is indicated in a text window. You may view the files in the list in steps of 1, 10, or continously, by using the SingleStep, FastStep, and Cycle buttons, respectively. Additional convenience buttons let you jump to the beginning or end of the sequence. Radio buttons can be used to set the direction of the cycling.

When you like the cycle, you may press the Auto Snap button and wait for your sequence of PNG files to be created.

As you render subsequent frames, the rendering parameters may be slightly different from frame to frame. The reason is that the rendering parameters depend on the dimensions of the data set. You can switch off repeated udpateds of the view parameter settings by clicking the Keep ViewScale check box (see also section 8.2).

You may also render tracks that show the motion of single particles during the sequence; see section 7.2.
