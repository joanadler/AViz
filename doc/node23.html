Atoms...

This menu entry is active if the render mode is set to ''Atom''.

Click on Atoms/Molecules... to open the atom type dialog window to set some properties that affect the rendering of atoms of a given type, such as Al.

A selector is used to select the atom type to which the settings apply.

A toggle button labeled Show Atoms can be used to switch off the rendering of all atoms of the selected atom type.

A color bar shows the color, or spectrum of colors, that is currently used to render atoms of the given type.

If rendering is not disabled, you can adjust the size of the atoms of this type, relative to the default size. This is done using three radio buttons labeled Regular, Smaller and Larger.

The coloring of the atoms of the selected atom type can be controlled using the radio buttons labeled Type, Position, Property, and ColorCode.

If ''Type'' is chosen, all atoms of the selected type are colored the same color (which is set using the Set Color... button).

If ''Position'' is chosen, the atoms of the selected type are colored according to their coordinate values. Radio buttons labeled X, Y, and Z are used to select the reference coordinate axis. (The color scale that is used is set using the Set Color... button.)

If ''Property'' is chosen, the atoms of the selected type are colored according to their property values given in the input data file in addition to their coordinate values. Radio buttons labeled 1, 2, etc. are used to select the reference property. (The color scale that is used is set using the Set Color... button.) This feature can be used to visualize scalar particle properties such as kinetic energy that are given together with the particle position in the coordinate file.

If ''ColorCode'' is chosen, you assume complete control over the coloring by specifying property colums that are interpteted as red-green-blue color values, ranging from 0.0 to 1.0. By default, the first three property values are taken as red, green, and blue values, respectively. For instance, if the default settings are chosen and the entry of the first particle is 

Al  3.57.0 24.34 0.249		0.0	1.0	0.0
that particle is rendered using a strong green color.

The settings are effective after Apply or Done is pressed.

Click on the Bonds... button in the atom type dialog, or select the Bonds... submenu entry to open the bond dialog window. This window is used to control the rendering of bonds between atoms of given atom types, such as Al and O.

Two selectors, From Particle Type and To Particle Type, are used to select the atom types to which the settings apply.

Radio buttons labeled Present and Absent are used to enable and disable the rendering of bonds between the atoms of the specified types.

If bond drawing is enabled, the color of the bond can be set using the radio buttons in the Color box.

The thickness can be set using the radio buttons in the Thickness box. If Line is chosen, the bonds will be rendered as lines; this is the fastest rendering mode. Other thicknesses, in combination with high render quality, will give you bonds that are rendered as cylinders.

Depending on the view point, the rendering can be improved by activating anti-aliasing. Use the checkbox labeled AntiAliasing to control this feature.

A text window labeled Emission is provided to set the emission light quality of the bond. In certain circumstances bond light emission can improve the visual appearance of the rendering.

Most important are the minimum length and maximum length parameters, to be entered in text windows labeled Min Length and Max Length. The lengths are given in the same units as the particle coordinates. Only those bonds will be rendered whose length is within the limits specified.

The setting is effective after Apply or Done is pressed.
