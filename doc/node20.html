Set Default View Param...

Click on Set Default View Param to save the current view parameters in a view parameter file called aviz.vpm in a hidden directory .aviz in your home directory. If this file is present in the directory in which AViz is started the next time, AViz will automatically read and set these parameter values.

The view parameters include parameters such as view point, lighting, and annotation. They do not include parameters related to the rendering of a class of particles, such as the color of the 'Al' atoms. The idea is that while you may want to change the view point frequently, you may not want to change particle colors.
