Polymers...

The ''Polymer'' rendering mode is almost identical to the ''Atom'' rendering mode. The polymer type dialog that comes up when you choose the Polymers... submenu entry produces a dialog that is identical to the atom type dialog. The polymer bond dialog, popping up when you select the Bonds... submenu entry is also identical with the atom bond dialog.

The difference is that polymer bonds are only drawn between particles that belong to the same molecule. The fourth column in the coordinate file is expected to contain an integer value indicating, for each particle, the molecule to which it belongs. For instance, two atoms belonging to the first molecule may be entered as follows: 

C  0.0 0.0 0.0 1
H  0.5 0.0 0.0 1
